# BMSTU_graduation_work

Объектом исследований в данной работе является полет снаряда. Целью является моделирование полета снаряда. 
Работа подразумевает решение прямой и обратной задачи. 
Смысл прямой задачи состоит в получении траектории полета снаряда основываясь на начальной точке его полета.
Обратной задачи – по 4 точкам восстановить кривую и найти место выстрела и место падения. 
Для этого нужно определить коэффициенты разложения и изменять время.

Необходимо создать математическую модель полета снаряда с помощью программы на языке программирования СИ++.  
Также требуется организовать вывод данных в базу данных mysql с помощью программы на языке программирования python.
